How to Use the Offline Web Scraper

Extract structured JSON or CSV data from any website without writing Python scripts.

Step 1: Grab the Target HTML

Because this scraper runs offline in your browser, it cannot visit websites for you. You must provide it with the raw HTML code of the data you want to extract.

  • Go to the webpage you want to scrape.
  • Right-click near the data (like a grid of products or a list of articles) and select Inspect.
  • In the code panel that opens, hover your mouse over the lines of code until the browser highlights the entire area containing your items.
  • Right-click that line of code, select Copy ➔ Copy element (or Copy OuterHTML).
  • Paste this block of code into the "Input Target HTML" box in the Scraper tool.

Step 2: Set the Parent Selector (Optional but Recommended)

If you are scraping a list of items (like 10 different laptops on a store page), you need to tell the scraper how to separate them into rows.

  • Find the CSS class that wraps each individual item.
  • Example: If every laptop is inside a <div class="product-card">, type .product-card into the Parent Selector box.
  • Note: If you leave this blank, the scraper will search the entire HTML block as one single item.

Step 3: Build Your Extraction Rules

Click "Add Data Field" for every specific piece of information you want to pull out of each item. You need three things for a rule:

  • Field Name: What you want the column to be called (e.g., Price, Author, Image_URL).
  • CSS Selector: The exact HTML tag or class holding the data.
    Examples: h2, .price-tag, img.thumbnail, a.read-more
  • Data Type: Choose what part of the element to extract:
    • Text Content: Grabs the visible words (Best for titles, prices, descriptions).
    • Link (href): Grabs the URL from an <a> tag.
    • Image (src): Grabs the image link from an <img> tag.
    • Inner HTML: Grabs the raw HTML inside the element (Useful if you want to keep bold/italic formatting).

Step 4: Extract and Export

Once your rules are set, click the big green 🚀 Run Extraction button.

The tool will instantly parse the DOM and output perfectly formatted JSON data. You can then use the buttons below the output box to immediately download the results as a .json file or a spreadsheet-ready .csv file!
