💻 Writing Code Blocks (Important!)

Since you are building a developer suite, you will frequently need to save blocks of code. To do this, you use triple backticks (the key usually above the Tab key: `).

Inline Code

For a quick variable name, wrap it in single backticks:
`const myVar = 10;` becomes const myVar = 10;

Multi-line Code Blocks

To write a full block of code, you must explicitly open and close the block.

  1. Type three backticks followed by the language name (e.g., ```javascript or ```html).
  2. Press Enter and paste your code.
  3. Press Enter again and type three backticks ``` alone on the final line to close the block!

Example of a properly closed block:

```javascript
function calculateTotal(price) {
    return price * 1.2;
}
```

🛠️ Toolbar & Autosave

  • 💾 Save .md File: Downloads your raw text exactly as you typed it. Perfect for pushing to GitHub.
  • 📋 Copy as HTML: Copies the rendered preview. If you paste this into an email or a WordPress blog, it keeps all your bolding, lists, and code blocks perfectly formatted!
  • ✔️ Autosave Vault: Every time you press a key, your progress is saved to your browser's offline localStorage. If you accidentally close the tab, open it again and your text will be waiting for you.
