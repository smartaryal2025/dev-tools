Mastering the Code Workspace

The Code Workspace is a distraction-free, lightning-fast text editor. While it runs 100% offline in your browser, it is equipped with a custom engine that mimics the most powerful features of desktop IDEs like VS Code.

⌨️ VS Code Power Shortcuts

You don't need to reach for your mouse to manage your code. Use these built-in hotkeys to code faster:

  • Shift + Alt + ↓ Down Arrow : Duplicate Line. Instantly copies the current line (or highlighted lines) directly below.
  • Ctrl + Shift + K : Delete Line. Instantly deletes the entire line your cursor is on.
  • Tab : Smart Indent. Highlights multiple lines? It will push them all forward. No text highlighted? It checks for Snippets first, then inserts 4 spaces.
  • Shift + Tab : Outdent. Pulls the current line (or highlighted lines) back by 4 spaces.

📂 File System & True Overwrite

Because this app uses modern browser APIs, it interacts with your hard drive much like a native app.

  • Open File: Opens your native OS file picker. It remembers your last folder.
  • Save (Overwrite): The first time you click this, the browser will ask permission to link to the file. Once granted, clicking Save will silently overwrite the file on your hard drive with zero popups!
  • Save As: Forces the "Save" dialogue box to open so you can create a new copy or rename the file.
