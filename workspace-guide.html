Mastering the Code Workspace

The Code Workspace is a distraction-free, lightning-fast text editor powered by Microsoft Monaco (the engine behind VS Code). It runs entirely in your browser but acts like a native desktop application.

📂 True File Management

This editor uses modern Browser APIs to interact securely with your hard drive, skipping annoying download popups.

  • 📂 Open File: Opens a native file picker. The editor automatically detects the file extension (e.g., .html or .js) and switches the syntax colors to match.
  • 💾 Save (Overwrite): The first time you click this, it asks for permission to link to the file. Afterward, it silently overwrites the file on your hard drive so you can code seamlessly.
  • 📝 Save As: Forces the Save dialogue box to open so you can create a copy or rename your work.

🚀 Advanced Monaco Features

Because this uses the real VS Code engine, you have access to powerful professional tools natively:

  • The Command Palette (F1): Press F1 at any time to open the Monaco Command Palette. From here, you can search for and execute hundreds of hidden editor commands.
  • Multi-Cursor Editing: Hold Alt and click anywhere in your code to place multiple cursors. You can type, delete, or paste on dozens of lines simultaneously!
  • The Minimap: The right side of the editor features a zoomed-out visual map of your entire file, allowing you to click and drag to scroll through massive files instantly.
  • Code Folding: Hover over the line numbers on the left. You will see small arrows that allow you to collapse (fold) large functions, HTML divs, or CSS blocks to keep your workspace clean.

⌨️ Power Shortcuts Cheat Sheet

Keep your hands on the keyboard. Use these native shortcuts to write code at lightning speed:

Shortcut Action
Ctrl + S True Save: Silently overwrites the current file.
Ctrl + D Add Selection to Next Find Match: Highlights the current word. Press it again to highlight the next instance of that word so you can edit both at once!
Ctrl + / Toggle Comment: Instantly comments or uncomments the current line (or selected block) using the correct syntax for your language.
Shift + Alt + ↓ Copy Line Down: Duplicates the current line directly below.
Alt + ↑ / ↓ Move Line: Moves the current line of code up or down through your file.
Ctrl + F Find & Replace: Opens the native search widget in the top right corner.
Tab / Shift+Tab Indent / Outdent: Pushes or pulls the current line (or all highlighted lines) by 4 spaces.
